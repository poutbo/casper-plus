casper-plus
===========

Ghost-Theme

对Ghost的默认模板进行了一些修改.

1. 修改样式外观，头部改为黑色
2. 调整Goole字体服务器可配置
3. 增加disqus引用

配置方式:
修改`post.hbs`和`partials/loop.hbs`将`likejs-com`更换为你的disqus配置
如想更换Google字体服务器 请修改`defult.hbs`将`useso`修改为你需要的.
